# Smart Project

This is a new Python project.

## Setup

1.  Activate the virtual environment:
    *   Windows: `.venv\Scripts\activate`
    *   Mac/Linux: `source .venv/bin/activate`
2.  Install dependencies (if any):
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the main script:
```bash
python main.py
```
